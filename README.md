# Smart Management Certificate Registry

Сайт для выдачи сертификатов с QR-кодом и публичной проверки через реестр.

## Роли

- Ученик открывает QR-ссылку или вводит номер сертификата и видит только данные своего сертификата.
- Учитель входит в закрытый кабинет, создает сертификаты, скачивает PDF через печать браузера и видит полный реестр.

## Как открыть

Откройте `index.html` в браузере для локального просмотра или разместите всю папку на PHP-хостинге.

Для нормальной защиты нужен PHP-хостинг, потому что пароль и права доступа проверяются в `api.php`.

## Пароль учителя

По умолчанию пароль: `teacher2026`.

На хостинге лучше задать переменную окружения `TEACHER_PASSWORD` и заменить пароль на свой.

## Как работает

- Публичная проверка сертификата: `api.php?id=SM-2026-0001`.
- Закрытый реестр учителя: `api.php?action=list`, только после входа.
- Создание сертификата: `POST api.php` с `action: "create"`, только после входа.
- Сертификаты сохраняются в `data/certificates.json`.
- QR-код ведет на страницу вида `index.html#certificate/SM-2026-0001`.
- Кнопка `Вернуться в главное меню` открывает `https://smart-management.uz/`.
- Кнопка `Скачать PDF` открывает печать сертификата. В окне браузера выберите `Сохранить как PDF`.

## Важно перед публикацией

Папке `data` нужны права на запись для PHP. Если хостинг не дает PHP записывать JSON-файл, новые сертификаты не будут сохраняться на сервере.
